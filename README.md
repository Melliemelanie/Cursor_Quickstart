# Cursor IDE Setup Tool

A web application that generates one-click setup scripts for Cursor IDE projects.

## Features

- **Project Type Selector**: Choose from React, Node.js, Next.js, or Custom presets.
- **Automated Script Generation**: Generates Bash (macOS/Linux) and PowerShell (Windows) scripts.
- **One-Click Setup**: Scripts handle directory creation, dependency installation, and configuration.
- **Visual Guide**: Step-by-step instructions and troubleshooting tips.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.

## Usage

1. Select your desired project type.
2. Enter a project name.
3. Click "Generate".
4. Download the script or copy it to your clipboard.
5. Run the script in your terminal to set up your new project.
